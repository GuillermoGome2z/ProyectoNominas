# 🏢 Sistema de Gestión de Nóminas | .NET 8 + Blazor

![.NET Version](https://img.shields.io/badge/.NET-8.0-512BD4?logo=dotnet)
![Blazor](https://img.shields.io/badge/Blazor-WebAssembly-512BD4?logo=blazor)
![License](https://img.shields.io/badge/License-MIT-green)

Sistema web completo para la administración de nóminas, recursos humanos y expedientes digitales.

## 🌟 Características Principales

- **Gestión de empleados** (CRUD completo con información laboral y personal)
- **Cálculo automático de nóminas** (IGSS, IRTRA, bonificaciones)
- **Expedientes digitales** (Subida/validación de documentos)
- **Reportes** (PDF/Excel con historial de pagos)
- **Autenticación segura** (JWT con roles: Admin, RRHH, Empleado)

## 🛠️ Stack Tecnológico

| Componente       | Tecnología                  |
|------------------|-----------------------------|
| Frontend         | Blazor WebAssembly          |
| Backend          | .NET 8 Web API              |
| Base de Datos    | SQL Server / MySQL          |
| ORM              | Entity Framework Core       |
| Autenticación    | JWT + OAuth2                |
| Estilos          | Bootstrap 5 + Tailwind CSS  |
